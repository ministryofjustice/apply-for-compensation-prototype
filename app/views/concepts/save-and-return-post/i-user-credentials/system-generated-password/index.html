If you don't complete your application in this session you'll need these pieces of information to return to it later:

Your save and return reference: HDJ2123F

Your three-word password: threerandomwords

This information won't be sent to you so you'll need to write it down. You can't reset or find out this information if you lose it, so you should keep it safe.

Saving your application

Your application progress is saved automatically every time you click Continue. You can close your application at any time and continue it later using your save and return reference and password above.

Completing your application if you stop

You can resume your application by visiting the Claim compensation if you were the victim of a violent crime page on gov.uk, selecting "Start a claim", and selecting "Resume a saved application".

Saved application expiry

If you don't return to your saved application and complete it within 60 days, it will be deleted for your security.
