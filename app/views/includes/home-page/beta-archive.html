Complete self journey (UK + EU Citizens) 14 January 2021

Changes in this release

This snapshot is what went into the deployment which went out in January 2021. This is the service we put into pilot with some key changes that were made based on research.

  • Changed the violent crime question to a radio button list of incidents
  • Ask the applicant if they want to provide a description of the incident
  • Ask the applicant if they want to provide any additional Information
  • The declaration was changed to rectify the inconsistency in the medical consent text
  • The Your Treatment context page was updated
  • Timeout timeframe was increase (but that is not active on the prototype)

Pilot for physical injuries and financial losses 26 to 29 Oct 2020

Changes in this release

In Nov 2020 we launched a pilot to validate the work that we had done on physical injuries, loss of earnings with real users.

  • Added a qualifying question for bereavement applications
  • Included other incident types with violent crime question
  • Created a section for injuries using one thing per page format
  • Injuries section includes questions about special payments
  • Created a section for treatment and medical help details
  • A new section about financial losses for loss of earnings and special expenses

The iterations of the flow are documented here: Flow iteration from private beta -> public beta.


Release 2: public beta 14 May 2020

Changes in this release

In May 2020 we went into public beta with the journey for VOSAAs. We made the following changes from research on the private beta and reommendations from the assessment.

  • Updated the content on the consent page
  • Removed the options page
  • Added question about how to send confirmation
  • Swapped British Citizen and Who are you applying for questions
  • Added in context pages between key sections
  • Added in qualifying questions for injuries and financial losses
  • Added DMI questions
  • Added medical details questions
  • Changed the order of the About the crime questions
  • Updated content on Contact with the offender questions
  • Improved other compensation questions
  • Moved the declaration to the end and updated content
  • Changed groupings on CYA page
  • Updated content on Confirmation screen

The iterations of the flow are documented here: Flow iteration from private beta -> public beta.
